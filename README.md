## Topics Covered
- GPIO configuration
- Timer configuration
- interrupts
- UART communication
- Basic peripheral register programming
- Basic DMA Buffer
- Basic ADC configuration
## Tools
- MCU: STM32 (F407VET6)
- Language: C
- IDE: STM32CubeIDE
- Build system: GCC ARM Toolchain

## Goal
This repository is intended for **learning embedded systems fundamentals** and understanding how microcontrollers operate without abstraction layers.
